# Jauth
JWT RAILS AUTHENTICATION

## Usage
If you want to use spec, please remove `sample` from `database.sample.yml` and `storage.sample.yml`.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'jauth'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install jauth
```
